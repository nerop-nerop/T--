# T-Библиотека: Консольное приложение для управления личной библиотекой

## Описание проекта

"T-Библиотека" --- это консольное приложение, разработанное на языке
Python, предназначенное для эффективного управления личной коллекцией
книг. Приложение позволяет пользователям добавлять, просматривать,
искать, отмечать статус книг (прочитана/не прочитана), добавлять в
избранное и удалять книги. Все данные надежно сохраняются в файл при
завершении работы и автоматически загружаются при следующем запуске.

## Основные возможности

-   **Добавление книги:** Ввод названия, автора, жанра, года издания и
    краткого описания.
-   **Просмотр книг:** Отображение полного списка книг с возможностью
    сортировки (по названию, автору, году) и фильтрации (по жанру,
    статусу).
-   **Добавление в избранное:** Пометка книг как избранных для быстрого
    доступа.
-   **Изменение статуса книги:** Отметка книги как "прочитана" или "не
    прочитана".
-   **Удаление из избранного:** Снятие статуса "избранное" с книги.
-   **Просмотр избранных книг:** Отображение списка книг, отмеченных
    пользователем как избранные.
-   **Удаление книги:** Полное удаление книги из библиотеки.
-   **Поиск книги:** Поиск по ключевым словам в названии, авторе или
    описании.
-   **Сохранение и загрузка данных:** Автоматическое сохранение
    библиотеки в файл (`library_data.json`) при выходе и ее
    восстановление при запуске.

## Технологии

-   **Язык:** Python 3.x
-   **Сохранение данных:** JSON (для простой сериализации и
    десериализации)

## Инструкции по запуску

1.  **Убедитесь, что у вас установлен Python 3.x.**

2.  **Запустите приложение:** `bash     python main.py`

3.  **Следуйте инструкциям в консоли:** Приложение предложит меню с
    различными действиями. Вводите соответствующие цифры для выбора
    желаемой операции.
